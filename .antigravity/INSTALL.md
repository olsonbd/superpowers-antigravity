# Superpowers for Antigravity

## Installation

You have already cloned the repository.

To use the tools, you can run them directly from the `.antigravity` directory.

## Usage

### Bootstrap

Run this to see instructions and available skills:

```bash
./.antigravity/superpowers-antigravity bootstrap
```

### Find Skills

```bash
./.antigravity/superpowers-antigravity find-skills
```

### Use Skill

```bash
./.antigravity/superpowers-antigravity use-skill <skill-name>
```
